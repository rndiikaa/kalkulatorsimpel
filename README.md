# kalkulatorsimpel
Program Kalkulator Sederhana adalah aplikasi berbasis Python yang digunakan untuk melakukan operasi aritmatika dasar, yaitu penjumlahan, pengurangan, perkalian, dan pembagian. Program ini dilengkapi dengan menu interaktif sehingga pengguna dapat memilih jenis operasi yang diinginkan melalui input angka.
